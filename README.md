### Проект автоматизации тестирования функций интернет-магазина по курсу со Stepik.

Ссылка на курс: https://stepik.org/course/575/syllabus

⚠️ **Внимание!** Если у вас не запускаются автотесты, пожалуйста уберите точки из импортов в файлах с тестами, всё должно запуститься! ⚠️

Тесты написаны на языке Python с использованием Selenium, Pytest, и паттерна Page Object Model. Данный проект содержит автотесты для проверки функциональности интернет магазина, а именно:

> Добавление в корзину

> Регистрация нового пользователя и проверка авторизации

> Видимость элементов на каждой странице сайта, а именно Корзины и ссылки на логин/регистрацию

> Переходы из различных страниц сайта в корзину и страницу логина/регистрации
